## 使用方法：
在群聊内输入`#方舟十连` 或者`#方舟单抽`即可

## 如何修改：
直接修改arknights.lua里的`starX_name`（X代表星数），数据使用json格式，最后会自动计算数量

## 随机逻辑：
先根据星数对应的权重随机出本次的星数，之后根据星数随机出对应的干员
> 本随机算法可能与官方算法有所不同，在此仅做娱乐项目！
